### Install notes
The setup as of 09/17/2018.
- Termite with zsh, which means when using ssh a termite terminfo file should be copied to remote.
- To automatically lock the screen when suspending, enable i3lock.service which uses the lock script located in config/i3/scripts.
- [Zplug](https://github.com/zplug/zplug) is used as package mananger for zsh
- [Dein](https://github.com/Shougo/dein.vim) is used as package manager for nvim
